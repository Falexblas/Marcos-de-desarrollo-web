Curso: Pruebas unitarias con Java (JUnit y Mockito)

Asegura la calidad de tu software con Java, JUnit y Mockito.

¿Qué aprenderás?

  • Se explorarán los fundamentos y beneficios de usar JUnit junto con Mockito para realizar pruebas unitarias y comprobar la funcionalidad del código de una aplicación.
  • Aprenderás a crear mocks para simular elementos externos al método o clase que estás probando.
